Step 4b: Variable Selection Sensitivity Analysis

Dermacentor variabilis Species Distribution Model

Author

Alexander W. Gofton

Published

April 7, 2026

Overview

The primary analysis forced retention of Bio1 (annual mean temperature), which resulted in VIF values up to 47.25 and a 66.4% MaxEnt contribution that masked the importance of other variables. Bio1 encodes cold-winter gradients in eastern North America that are ecologically irrelevant to Australian projection.

This script defines three alternative variable sets for sensitivity testing:

Set Variables Rationale
A (current) bio1, bio5, bio7, bio10, bio12, bio15, aridity_idx, PET_seasonality Baseline with Bio1 forced
B (no Bio1) bio5, bio7, bio10, bio12, bio15, aridity_idx, PET_seasonality Remove dominant collinear variable
C (moisture-focused) bio5, bio10, bio12, bio15, aridity_idx, PET_seasonality, vpd_annual Replace Bio1 + Bio7 with VPD

Set C aligns with the CLIMEX finding (Halliday & Sutherst 1990) that low humidity is the primary limiting factor for D. variabilis in Australia.

3. VIF analysis for each set

# Function to compute VIF for a variable set
compute_vif <- function(data, vars, set_name) {
  cat(sprintf("\n--- VIF for %s ---\n", set_name))
  env_cols <- data %>%
    dplyr::select(all_of(vars)) %>%
    as.data.frame() %>%
    na.omit()

  vif_vals <- vif(env_cols)
  print(vif_vals)

  max_vif <- max(vif_vals$VIF)
  cat(sprintf("Maximum VIF: %.2f %s\n",
              max_vif,
              ifelse(max_vif > 10, "*** EXCEEDS THRESHOLD ***", "(OK)")))

  return(vif_vals)
}

4. Pairwise correlations for each set

check_correlations <- function(data, vars, set_name) {
  env_cols <- data %>%
    dplyr::select(all_of(vars)) %>%
    as.data.frame() %>%
    na.omit()

  cor_mat <- cor(env_cols)
  high_cor <- which(abs(cor_mat) > 0.7 & upper.tri(cor_mat), arr.ind = TRUE)

  cat(sprintf("\n--- High correlations (|r| > 0.7) for %s ---\n", set_name))
  if (nrow(high_cor) > 0) {
    for (i in seq_len(nrow(high_cor))) {
      v1 <- colnames(cor_mat)[high_cor[i, 1]]
      v2 <- colnames(cor_mat)[high_cor[i, 2]]
      r <- cor_mat[high_cor[i, 1], high_cor[i, 2]]
      cat(sprintf("  %s ~ %s: r = %.3f\n", v1, v2, r))
    }
  } else {
    cat("  None — all pairwise |r| < 0.7\n")
  }
}

check_correlations(occ_env, set_A, "Set A")
